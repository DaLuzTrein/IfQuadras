# IfQuadras

## About
This project is a sports court rental management system developed as part of my Programming II subject in the Systems Development Course at IFSul. It was designed to practice PHP concepts and apply them to a real-world scenario, managing users, clients, and rentals.

## Features
- Admin login and management
- Creation of client accounts linked to user accounts
- Rental management for sports courts
- Rental management for equipments

## Technologies Used
- PHP
- MySQL
- HTML / CSS
